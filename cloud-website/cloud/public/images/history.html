History

The history of cloud services is a long and rich history. Without the concept and inception of these services many ideas and technologies we have today would not exisit. If you wish to read more about the history below is a short timeline. For more information please visit here.

Time Line

1963

Defense Advanced Research Project Agency(DARPA) asked MIT to design a computer that could support multiple users at the same time. This was the start of modern day cloud services.

1969

J.C.R Lickider a psychologist and computer scientist. Assisted DARPA in developing a primative version of the modern internet. This vision came to be called the "Intergalatic Computer Network" by Lickider.

2002

Amazon launched their web based retail service. First major business to use only 10% of capacity as a problem to be solved. This was do to the Cloud Computing Infrastructure Model giving them the ability to use capicities of computers more effieiently

2006

Amazon launches Amazon web services(AWS). Which allowed other companies to purchase Cloud services like storage, computation, and even virtual computers.
Google Launches Google Documents originaly called writely, and Google Sheets.

2007

Google, IBM, and Univeristies built server farms to assist with large scale projects. Project that required large pools of data, and fast processors. This lead to top college renting services from Google and IBM to expediate research projects.

2011

IBM launches SmartCloud, their alternative to AWS. While Apple launched the ICloud which was more a storage system for users to store personal information. Mircosoft ramps up cloud services advertising to gain the interest of the public.
