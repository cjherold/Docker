Advantages and disadvantages

Why should I use cloud computing?

Advantages:
1. Easy implementation
2. Accessability
3. No hardware required
4. Flexibility for growth
5. Efficient recovery.

Disadvantages:
1. No longer in control
2. May not get all features
3. Need to retain a back-up server
4. No redundancy
5. Bandwidth issues.
